

## Steps to setup locally :
- Make sure that you have [Node.js](https://nodejs.org/en/download/) installed on your machine

# Frontend
```
cd frontend
```
- Install dependencies
```
npm install
```
- Run `development server`
```
npm start
```


# Backend
```
cd server
```
- Install dependencies
```
npm install
```
- Run `development server`
```
npm start
```
